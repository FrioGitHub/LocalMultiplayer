Connection between clients using localhost.

This repository is part of a tutorial, if you want to access it: https://youtu.be/Z-FjfPlbHfE

# Tutorial

### Compile files
```bash
javac -d . com/multiplayer/*.java
```

### Execute Server
```bash
java com.multiplayer.Server
```
